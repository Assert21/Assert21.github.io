Hexo联动GitHub搭建个人博客

Hexo联动GitHub搭建个人博客

九月 16, 2021
  • -教程

Hexo X GitHub 联动确认！

1、安装必要的插件

## 1、git插件  https://www.git-scm.com/download/
## 2、node.js插件  https://nodejs.org/en/
## 3、npm插件  包含在node.js插件中
## 4、检测三种插件是否已成功安装：
    ### 1、win+R输入cmd并运行打开命令符窗口
    ### 2、输入：
        ####1、git --version
        ####2、node -v
        ####3、npm -v
    ### 3、若均出现版本号则已安装成功

2、搭建GitHub仓库

## 1、注册并登录GitHub后，点击个人头像旁的下三角，下拉菜单中选择Your repositories
## 2、点击绿色的new创建属于自己的仓库
## 3、在创建页面的Repository name栏中输入形同<username>.github.io，其余均非必填项
## 4、点击Create repository创建仓库
## 5、注意事项：repository name务必为<username>.github.io 关系到后期hexo导入

3、配置ssh密钥

## 1、原因：每次向GitHub提交代码都需要经过你的账号验证，在未配置ssh密钥的情况下，需要
## 每次都输入邮箱和密码，极为繁琐，配置ssh密钥能快速自动通过这一过程，加速代码更新
## 2、查看本机中是否已经存在密钥：
    ### cd ~/.ssh   #此命令在命令符窗口中运行
## 3、如果显示No such file则需要重新生成：
    ### ssh-keygen -t rsa -C "邮箱地址"
## 邮箱地址应为你的GitHub邮箱地址即<username>.github.io，随后ssh key将保存在C盘用户
## 目录的.ssh文件夹中，路径大致为C:\Users\<你的电脑用户名>\.ssh
## 4、右键.ssh文件夹中的id_rsa.pub文件，在打开方式中选择用记事本打开，Ctrl+A全选内容，
## Ctrl+C复制全部内容
## 5、在GitHub中点击右上角的头像，下拉菜单中选择Setting，在左侧菜单中选择SSH and GPG keys
## 在顶部右侧选择绿色的New SSH key来新建ssh密钥
## 6、随便起个title，在下方Ctrl+V粘贴你的ssh密钥
## 7、点击Add SSH key完成对ssh密钥的添加

4、测试ssh密钥与GitHub绑定是否成功：

## 打开命令符窗口，在其中输入：
    ### ssh -T git@github.com
## 若是显示Hi！……You have successfully…… 则已成功

5、绑定你的git和GitHub

## 1、双击图标打开git bash
## 2、在窗口中输入：
    ### git config --global user.name "....."
    ### git config --global user.email "...@...."   #其中的...都为你自己的用户名和邮箱

6、配置你的hexo

## 1、安装hexo：
    ### 1、打开git bash
    ### 2、输入：npm install -g hexo
## 2、建立一个新文件夹作为博客文件存放处
## 3、右键该文件夹，选择git bash并输入：
    ### hexo init  #此步为初始化hexo博客文件夹
## 4、博客文件夹初始化成功，其中应包含有：
    ### node_modules
    ### scaffolds
    ### source  #资源文件夹，内含有一个_posts文件夹，内含.md格式的博客文件
    ### themes  #主题文件夹，用于存放各种主题，默认主题为landscape
    ### .gitignore
    ### _config.yml  #接下来要编辑的配置文件
    ### db.json
    ### ……

7、测试本地化启动博客：

## 1、右键博客文件夹，选择git bash
## 2、在其中输入：
    ### hexo g #用于将source/_posts/xxx.md这一博客文件生成到自己的本地博客
    ### hexo d #用于启动本地博客服务器来访问博客，一旦关机或停止博客就无法访问
## 3、在浏览器中输入http://localhost:4000来访问本地博客

8、联动hexo与GitHub

## 1、进入博客文件夹，右键config.yml文件，用记事本打开
## 2、拉至最底部，将其最后的代码修改为：
    ### deploy:
        ### type: git
        ### repository: git@github.com:<username>/<username>.github.io.git
        ### branch: <...> #此处填的内容为你GitHub仓库里branch左侧显示的文字
        ### 通常为main或master
    ### <username>即此处填的为你的GitHub用户名
## 3、Ctrl+S保存文件
## 4、右键博客文件夹，选择git bash
## 5、输入：
    ### hexo cl  #清除博客缓存
    ### hexo g  
    ### hexo d #将本地博客的文件上传至GitHub

9、完成与debug

## 1、打开http://<username>.github.io访问你自己的博客
## 2、若第8步的5中代码出现报错，请检查代码是否正确及空格是否正确
## 3、若短期内GitHub博客未成功更新，请通过shift+F5刷新浏览器缓存
## 4、完成后就可以考虑对自己的博客进行精装修了（指更换主题）
